O atributo de valor de entrada

O atributo value especifica um valor inicial para um campo de entrada:






O atributo somente leitura especifica que um campo de entrada deve ser somente leitura (não pode ser alterado):


O atributo disabled especifica que um campo de entrada deve ser desativado (inutilizável e não clicável):


O atributo input maxlength especifica o número máximo de caracteres permitidos em um campo de entrada.

O atributo size especifica a largura visível, em caracteres, de um campo de entrada.


O atributo placeholder especifica uma pequena dica que descreve o valor esperado de um campo de entrada.





O atributo required significa que é obrigatório informar os dados daquele campo

O atributo de foco automático de entrada

O atributo autofocus especifica que o campo de entrada deve receber foco automaticamente quando a página for carregada.
