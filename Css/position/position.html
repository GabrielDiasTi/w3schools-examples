posição: estática;

Um elemento com posição: estático; não está posicionado de forma especial; isto é o padrão do site:

Este elemento div tem position: static;

posição: relativa;

Um elemento com posição: relativa; está posicionado em relação à sua posição normal:

Este elemento div tem position: relative;

posição: fixa;

Um elemento com posição: fixo; está posicionado em relação à janela de visualização, o que significa que sempre permanece no mesmo lugar, mesmo se a página for rolada:

Este elemento div tem position: fixed;

posição: absoluta;

Um elemento com posição: absoluto; é posicionado em relação ao ancestral posicionado mais próximo (em vez de posicionado em relação à janela de visualização, como fixo):

Este elemento div tem position: relative;
Este elemento div tem a posição: absolute;

Tente rolar dentro desse quadro para entender como funciona o posicionamento fixo.
