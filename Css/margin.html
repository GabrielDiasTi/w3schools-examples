CSS Margins

As propriedades de margem CSS são usadas para criar espaço ao redor dos elementos, fora de quaisquer bordas definidas.

Este elemento tem uma margem de 70 px.

Usando propriedades de margem individuais

Este elemento div tem uma margem superior de 100px, uma margem direita de 150px, uma margem inferior de 100px e uma margem esquerda de 80px.

Margin:auto

Você pode definir a propriedade margin para auto para centralizar horizontalmente o elemento em seu contêiner. O elemento então ocupará a largura especificada e o espaço restante será dividido igualmente entre as margens esquerda e direita:

Este div será centralizado horizontalmente porque tem margem: auto;
