CSS Overflow

Configurando o valor de estouro para rolar, o estouro é cortado e uma barra de rolagem é adicionada para rolar dentro da caixa. Observe que isso adicionará uma barra de rolagem horizontal e verticalmente (mesmo se você não precisar dela):

You can use the overflow property when you want to have better control of the layout. The overflow property specifies what happens if content overflows an element's box.
