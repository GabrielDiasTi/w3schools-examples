div3
div4 - Aqui, claro: esquerda; move div4 para baixo, abaixo do div3 flutuante. O valor "left" limpa os elementos flutuou para a esquerda. Você também pode limpar "certo" e "ambos"

Clearfix

Neste exemplo, a imagem é mais alta do que o elemento que a contém e é flutuante, por isso transborda para fora de seu contêiner:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus imperdiet, nulla et dictum interdum...

Adicione uma classe clearfix com overflow: auto; para o elemento que contém, para corrigir este problema:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus imperdiet, nulla et dictum interdum...
